# Diabetes Prediction System

## Overview
This project uses Machine Learning to predict the onset of diabetes based on diagnostic measurements. The model is trained on the Pima Indians Diabetes Database.

## Features
- Pregnancies
- Glucose Level
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age
